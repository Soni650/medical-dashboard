# 🏥 MediCore — Medical Dashboard

A full-stack medical dashboard with React + Node.js/Express + SQLite.

## 🚀 Quick Start

### Step 1 — Install Dependencies
cd medical-dashboard
npm run install-all

### Step 2 — Start the App
npm start

- Backend  → http://localhost:5000
- Frontend → http://localhost:3000
